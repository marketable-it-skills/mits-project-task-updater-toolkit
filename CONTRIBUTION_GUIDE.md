# MITS Contrubution Guide

A MITS project egy open source szellemiségű projekt, ami nyitott az együttműködésre bárki számára, aki elfogadja a MITS elveit és szabályait és rendelkezik azzal a szakmai háttérrel, ami lehetővé tesz számára, hogy valódi hozzáadott értéket tudjon nyújtani a projekthez.

Ez a guide részletesen leírja az együttműködés lehetőségét, formáit és módjait.

A MITS-hez az elábbi formákban lehetséges jelenleg a hozzájárulás:

1. Saját fejlesztésű és szellemi tulajdonú projektfeladatok (pl. nemzeti vagy felkészülési versenyek modulfeladatai) átdolgozása és feljaánlása a közösség számára.

2. MITS-tutorial készítése és felajánlása a közösség számára.

3. A MITS-ben levő projektfeladatok és tutoriálok javítása.

## Saját fejlesztésű és szellemi tulajdonú projekt feladatok átdolgozása és a MITS rendszerbe illesztése

### Milyen feladatok kerülhetnek be a MITS-be?

Alapvetően olyan feladatok ajánlhatók fel a MITS projekt számára, amelyek

- szellemi tulajdoni jogával a felajánló teljes mértékben rendelkezik
- szellemiségében, stílusában követi a skills versenyek (pl. EuroSkills és WorldSkills) modulfeladatokat (test project-eket)
- harmonizál az érintett skill terület (pl. web technology) WorldSkills vagy EuroSkills követelményrendszerével (Technical Description)
- az aktuális munkakhelyi kihívásokat jól szimuláló projektfeladatotot ír le
- alkalmas lehet az IT szakképzésben egy adott szakmai terület készségeinek elsajátítására vagy gyakorlására

Különösen szívesen várjuk a WorldSkills sztendernek megfelelő nemzeti válogatók és tréningversenyek feladatait.

### A feladatok átdolgozása

#### Milyen feltételeknek kell megfelelnie a felajánlott projektfeladatnak?

Ahhoz, hogy a felajánlott projektfeladat bekerülhessen a MITS-be, szükséges azt olyan formára hozni, ami megfelel a MITS Test Project Guide előrásainak, azaz

- a megadott folder és fájl struktúrát és elnevezéseket használja
- a projektfeladat leírásának (`project-description.md`) felépítése és belső szerkezete megfelel a MITS Test Project Guide-ban részletezett követelményeknek
- rendelkezik részletes, a MITS Test Project Guide-ban leírt formátumú marking scheme-mel (`marking-scheme.json`)
- a `metadata.json` tartalmazza a MITS Test Project Guide-ban leírt összes metadatot és megfelel a MITS Test Project Guide-ban leírt formátumnak.

#### A feladatok átdolgozásának folyamata

A feladatok átdolgozása megvalósulhat hagyományos módszerekkel is, de ajánlott a MITS Test Project Updater AI megoldásokkal támogatott eszközeit használni.
